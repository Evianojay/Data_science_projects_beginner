# NYC Schools SAT Analysis

## Project Overview
This project analyzes SAT performance data from NYC schools using Python and Pandas.

## Objectives
- Identify schools with the best math results
- Find the top 10 schools based on total SAT scores
- Determine which borough has the largest SAT score variability

## Technologies Used
- Python
- Pandas

│
├── data/

## How to Run
1. Clone the repository
2. Create and activate a virtual environment
3. Install dependencies:

pip install -r requirements.txt

4. Run the script:

python analysis.py
